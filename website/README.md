# OvuMate Website

This is the official website for the OvuMate mobile application.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient design with smooth animations
- **Fast Loading**: Optimized for performance
- **SEO Friendly**: Proper meta tags and semantic HTML

## File Structure

```
website/
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## How to Use

1. Open `index.html` in a web browser
2. Or deploy to any web hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-pink: #E55A8A;
    --secondary-purple: #8B4DA6;
    --accent-teal: #00B894;
}
```

### Add Download Links
Edit the download buttons in `index.html` and add your APK download link in `script.js`.

### Add Content
Edit the sections in `index.html` to add or modify content.

## Deployment

### GitHub Pages
1. Create a new repository
2. Upload all files
3. Go to Settings > Pages
4. Select main branch and /root folder
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Netlify
1. Drag and drop the website folder to Netlify
2. Your site will be live instantly

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the website directory
3. Follow the prompts
