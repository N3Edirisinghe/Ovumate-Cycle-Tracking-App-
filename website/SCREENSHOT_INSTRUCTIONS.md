# App Screenshot Setup Instructions

## How to Add Your App Screenshot

1. **Save your app screenshot** as `app-screenshot.png` in the `website` folder
   - The image should show the actual OvuMate app interface
   - File location: `new cycle/website/app-screenshot.png`

2. **Image Requirements:**
   - Format: PNG (recommended) or JPG
   - Recommended size: 320x640px (or similar mobile aspect ratio)
   - The image will be automatically scaled to fit the phone mockup
   - Should show the full app interface clearly

3. **File Structure:**
   ```
   website/
   ├── index.html
   ├── style.css
   ├── script.js
   ├── logo.png
   └── app-screenshot.png  ← Place your screenshot here
   ```

## Screenshot Display

The screenshot will automatically:
- Fit within the phone mockup (320x640px)
- Maintain aspect ratio
- Display with proper scaling
- Show the full interface

## Image Optimization Tips

- Use PNG format for better quality
- Use JPG format for smaller file sizes
- Optimize the image to reduce file size (recommended: under 500KB)
- Ensure the screenshot shows the main app interface clearly
- Recommended dimensions: 320x640px or 375x812px (iPhone size)

## If Screenshot is Missing

If the screenshot file is not found, a placeholder message will be displayed instead.

## Testing

1. Save your app screenshot as `app-screenshot.png` in the website folder
2. Open `index.html` in a web browser
3. The screenshot should appear inside the phone mockup in the hero section
